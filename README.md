## 📋 Descrição do Projeto
O Sistema de Reservas de Espaços está a ser desenvolvido para facilitar o funcionamento das aulas e ajudar os professores a encontrar rapidamente espaços alternativos, como salas de informática ou outros espaços, para outras atividades que os professores possam querer fazer.

## 🎯 Objetivos Principais
- [ ] Desenvolver um site dinâmico com base de dados
- [ ] Implementar sistema de autenticação de utilizadores
- [ ] Criar interface responsiva e intuitiva
- [ ] Implementar funcionalidade CRUD completa
- [ ] Fazer deploy da aplicação
